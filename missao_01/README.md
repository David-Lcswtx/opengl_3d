# Missão 1 – A Torre do Vórtice

Seu primeiro desafio como aprendiz no mundo 3D é materializar a Torre do Vórtice, uma estrutura em formato de cone, vista de três pontos cardeais distintos. Você deve usar glutWireCone() para renderizar o cone. Altere a posição da câmera (ex: com gluLookAt) para visualizá-la de três ângulos diferentes: frontal, lateral e superior. Marque cada ângulo com uma cor diferente de fundo para distinguir as perspectivas.
